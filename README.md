# Unit Deliverable 2 - Final Project GUI

## Project Description
My final project will be a **Task Manager Application** created from my `Task` class project. The program allows users to create, edit, organize, and delete tasks using a graphical user interface.

Each task contains:
- **Title** – task name  
- **Priority** – importance level  
- **Description** – task details  
- **Category** – School, Work, Personal, etc.

The goal of this project is to help users stay organized and manage daily responsibilities efficiently.

## Features
- Add new tasks
- Update task information
- Delete tasks
- View all saved tasks
- Organize by category
- Set priority levels
- Easy-to-use GUI
